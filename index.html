Mary and Tracy's Progress Report

Project Descrption

I (Tracy Pham) am working with Mary Duong. We are creating a platform game where each platform has random width and height, and the player's character must progress in the game by jumping to each platform. The platforms will be created using 2D real-time graphic using WebGL. If the character falls in between the gaps, they will die and its game over. The game will run for a short amount of time, so as long as they prgoress through all the platforms and make it to the end, they win. The character will be controlled by an audio input, via the player's microphone. No input is no movement. Low volume is moving forward, and high volume is jumping. Voice processing will be done using the p5.js library.

Progress Plan

One of our milestone is to learn how to use WebGL and how to use P5.js voice processing. Another milestone is to create the game itself, then implement voice control to move the player through the game.
